# 오브젝트풀(어드레서블)

목적 : 어드레서블을 활용한 게임내에 사용할 오브젝트 재활용
특징 : 
라이프 타임으로 관리하여, 오래된메모리 정리기능 추가.
오브젝트 풀안에 객체들의 라이프 타임을 볼수있는 툴 추가. 
테스트 모드를 사용 시 히어라키 자동 정리 기능 추가.
* 테스트 모드는 편의를 위해 bool 작성 필요하다면 전처리로 변경
